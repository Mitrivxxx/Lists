Detailed Description

template<class T>
class DoublyLinkedList< T >

Klasa implementujaca liste dwukierunkowa.

Ta klasa umozliwia operacje na liscie dwukierunkowej, takie jak dodawanie, usuwanie i wyswietlanie elementow w obu kierunkach.

Template Parameters
TTyp elementow przechowywanych w liscie.
